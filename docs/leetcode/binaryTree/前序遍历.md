## 二叉树的前序遍历

题目描述：给你二叉树的根节点 root ，返回其节点值的前序遍历 。 （即逐层地，从左到右访问所有节点）。

![示例](/二叉树层序遍历.png)

**example1:**

> 输入：root = [3,9,20,null,null,15,7]
>
> 输出：[[3],[9,20],[15,7]]

**example2:**

> 输入：root = [1]
>
> 输出：[[1]]

**example3:**

> 输入：root = []
>
> 输出：[]
